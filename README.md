## Plants:

### Get all plants for user
**_[GET]_** /api/:id/plants


### Get individual plant
**_[GET]_** /api/:id/plants/:plantId

### Add a Plant
**_[POST]_** /api/:id/plants

### Edit a Plant
**_[PUT]_** /api/:id/plants/:plantId

### Delete a Plant
**_[DELETE]_** /api/:id/plants/:plantId
